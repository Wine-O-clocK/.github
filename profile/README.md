  <h1>Wine'OclocK</h1>
  <p> 와인 입문자를 위한 와인 추천 웹 서비스. 코로나 이후 홈술 문화가 정착이 되면서 와인에 대한 관심이 매우 증가했으며, 와인을 소비하는 연령층도 낮아지고, 더이상 와인 전문샵에 가지 않고도 편의점에서 와인을 구매할 수 있을 정도로 위치 접근성이 높아졌습니다. 와인에 대한 관심도는 높아졌지만 여전히 와인에 대한 심적 접근성은 높다. 이에 와인 입문자들이 쉽고 간편하게 와인을 마실 수 있도록 와인 입문자를 위한 와인 추천 웹서비스를 기획하게 되었으며, 추천 알고리즘을 기반으로 추천 기능을 구현했습니다. 이를 통해 와인 입문자들이 더욱 와인에 대한 접근성을 높힐 수 있게 했습니다. </p>

2022년 9월 ~ 2022년 12월

</br>

## 🍷 Description
와인입문자들을 위한 와인 추천 및 검색 서비스.
- 크롤링 기반의 와인 추천 기능
  - 언급이 많은 와인
  - 접근성이 좋은 와인
  - 선물하기 좋은 와인
  - 가성비가 좋은 와인
  - 최신 언급 와인
  - 월 별 이벤트 와인
- 추천 알고리즘 기반의 와인 추천 기능
  - lv1 : 콘텐츠 기반 추천 알고리즘
  - lv2 : 아이템 기반 추천 알고리즘
  - lv3 : 잠재 요인 협업 기반 추천 알고리즘
- 상황에 맞는 와인들을 골라 추천
  - SNS 공유 가능 
- 와인 검색 기능
  - 키워드로 와인 이름 검색 가능
  - 필터링을 통해 와인 검색 가능
- 와인 기록 기능
  - 와인 저장하기
  - 와인 테이스팅 점수 부여
  - 와인 테이스팅 노트 작성

## 🍷 Feature
- 검색기능
  - 필터링 검색 : 필터링을 통해 조건에 맞는 와인 검색
  - 키워드 검색 : 특정 키워드가 들어간 와인 검색
- 테스트기능
  - 챗봇과 대화하며 상황에 맞는 와인을 테스트
- 와인 저장 및 기록
  - 하트 버튼을 통해 와인 저장가능
  - 테이스팅 노트로 점수와 짧은 리뷰 작성 가능
- 와인 추천받기
  - 유저 레벨에 따라 다르게 추천 알고리즘이 다르게 적용
  - 콘텐츠 기반, 아이템 기반, 잠재요인 기반 추천 알고리즘을 통한 와인 추천
